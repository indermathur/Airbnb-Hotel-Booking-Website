🏡 Airbnb Clone (Full Stack)

A full-stack Airbnb clone web application where users can browse listings, view property details, and interact with the platform just like a real booking system.

🚀 Features

🏠 Browse different property listings

🔍 View detailed property information

👤 User authentication (Login / Signup)

📅 Booking system with calendar

⭐ Reviews & Ratings

➕ Add new property listings

❤️ Save / Wishlist functionality (if added)

📦 Full-stack architecture (Frontend + Backend)

🛠️ Tech Stack

Frontend:

React.js

Tailwind CSS / CSS

Axios

Backend:

Node.js

Express.js

Database:

MongoDB (Mongoose)

📂 Project Setup

Follow these steps to run the project locally 👇

1️⃣ Clone / Download the Project
git clone <your-repo-url>

OR download ZIP and:

👉 Extract (Unzip) the project folder

2️⃣ Setup Environment Variables

Go to the backend folder

Create a .env file

Add your credentials like:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
3️⃣ Install Dependencies

👉 Open terminal in both folders:

Backend:
cd backend
npm install
Frontend:
cd frontend
npm install
4️⃣ Run the Project

👉 Start both frontend and backend:

Backend:
npm run dev
Frontend:
npm run dev
